# SynthesizerDatapack

## Guide

For information about how this datapack works and the items it adds, see the [Wiki](https://github.com/Sammcb/SynthesizerDatapack/wiki).

## Development

### Building

To generate a zipfile of the datapack, run `make`. To cleanup build artifacts, run `make clean`.

### Linting

To lint changes to the project, run `make lint`.
